# **App Name**: YardDesk

## Core Features:

- Inbox Management: Aggregate and display incoming requests or notifications.
- Ticket Management: Create, view, and update tickets with detailed information and status tracking at /ticket/[id].
- Customer Profiles: Maintain profiles for customers, accessible via /customers/[id].
- Template Management: Create and manage templates for common documents or responses, available at /templates.
- Analytics Dashboard: Display key performance indicators and insights at /analytics.
- Settings Panel: Allow users to configure application settings, theming and preferences via /settings.
- Global Search: Enable searching across all modules from a global search bar.

## Style Guidelines:

- Dark color scheme.
- Primary color: Slate (#708090) for a neutral and professional base.
- Accent color: Emerald (#50C878) for highlights and call-to-actions, in accordance with the request.
- Font: 'Inter' sans-serif for clear, modern text in headlines and body.
- Left sidebar navigation for primary routes.
- Top bar with global search functionality.
- Content area using cards and tables for data presentation.
- Lucide-react icons throughout the interface.
- Subtle transitions and animations for enhanced user experience.
